Q3:what is the difference between <span> and <div> tags?
Ans:<div> is used for bigger sections, while <span> is used for small inline content.
Example:
<div>This is div tag</div>
<span>This is span tag</span>

Q4:what is the difference between Block-level elements and inline elements?
Ans:Block-level elements take full width and start from a new line.
Inline elements only take required space and stay in the same line.

Ans:The DOCTYPE declaration it helps the browser display the webpage correctly.
Example:

Q6:What is the difference between id and class attributes?
Ans:The id attribute is used for a unique element, while the class attribute is used for multiple elements.
id can be used only once on a page.
class can be used many times.
Example:
<h1 id="title">Welcome</h1>
<p class="text">Hello</p>
<p class="text">World</p>

Q7:How do you create a from in HTML?Which input type are commonly used?
Ans:A form in HTML is created using the <form> tag.
It is used to collect user data like name, email, password, etc.
Example:
<form>
  <input type="text" placeholder="Enter Name">

  <input type="email" placeholder="Enter Email">

  <input type="password" placeholder="Enter Password">

  <input type="submit" value="Submit">
</form>

Q9:Explain the purpose of the alt attribute in the <img> tag?
Ans:The alt attribute in the <img> tag is used to show alternative text if the image does not load.
It also helps screen readers and improves SEO.
Example:
<img src="cat.jpg" alt="Cute Cat">
